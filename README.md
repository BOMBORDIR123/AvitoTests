## Этот проект содержит набор автоматизированных UI-тестов для веб-приложения Авито
## Avito Tech Internship Task Manager:
**https://avito-tech-internship-psi.vercel.app/**

## Приложение представляет собой систему управления задачами, где пользователь может:

- создавать задачи;

- редактировать задачи;

- просматривать карточки задач;

- искать задачи;

- фильтровать по статусам;

- работать с доской задач;

- перемещаться между разделами.

## Тестовый проект написан на Java + Playwright, использует JUnit 5, формирует Allure-отчёты
### Поддерживает запись фото и видео, позволяя их просматривать в Allure-отчёте, для быстрого нахождения ошибки
### Многопоточный запуск для ускорения проекта и выполнение тестов на разных окружениях.

### Проект протестирован в разных браузерах:

- Chromium

- Firefox

- WebKit

## Запуск проекта
➤ 1. Установка зависимостей

Перед первым запуском установите браузеры для Playwright:

mvn playwright:install

### ▶ Запуск тестов
Базовый запуск

Запуск всех тестов по умолчанию:

mvn test

### ▶ Запись видео тестов

Чтобы включить запись видео, добавьте параметр:

mvn test -DisVideoRecord=true

### Видео сохраняются в директорию:

build/video-results

### ▶ Выбор браузера для запуска

По умолчанию используется Chromium, но можно указать конкретный браузер:

- Chromium (по умолчанию)
 ```mvn test -Dbrowser=chromium```

- Firefox
```mvn test -Dbrowser=firefox```

- WebKit
```mvn test -Dbrowser=webkit```

## ▶ Запуск тестов + мгновенное открытие Allure отчёта (рекомендуется)

- Эта последовательность сразу откроет визуальный отчёт со скриншотами и видео:

```mvn test -DisVideoRecord=true```

```mvn allure:serve```

### Формирование Allure отчёта
1. Запуск тестов с генерацией Allure результатов:
   mvn clean test


После запуска Allure-файлы появятся в папке:
/build/allure-results

2. Просмотр отчёта:
   allure serve target/allure-results


Команда откроет интерактивный отчёт в браузере.

3. Генерация отчёта в директорию:
   allure generate target/allure-results -o target/allure-report --clean

4. Просмотр локального отчёта:
   allure open target/allure-report
